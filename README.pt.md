## ✨ Recursos

Esta skill permite gerar vídeos a partir de múltiplos inputs (imagem, vídeo, áudio, texto) diretamente no seu fluxo de trabalho OpenClaw.

- ✅ **Geração Controlável**: Utilize `@nome_do_material` para especificar papéis e movimentos de câmera.
- ✅ **Multimodal**: Suporte para imagem, vídeo e áudio como entrada simultânea.
- ✅ **Alta Qualidade**: Vídeos de até 720p com duração configurável.

---

## 🎮 Como Usar

Use `@nome_do_material` no prompt para definir a função de cada input.

```
@image1 como quadro inicial, @video1 como referência para a linguagem de câmera, @audio1 para música de fundo
```
